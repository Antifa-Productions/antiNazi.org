Privacy Policy

Last updated: July 20, 2026

This website operates under the principle that minimal data collection respects user autonomy. We do not track visitors, sell personal information, or employ third-party analytics. Our commitment to privacy aligns with our mission to provide public-domain radical literature freely and without surveillance.

Key Practices at a Glance

No Tracking

We do not use cookies, pixel tags, fingerprinting, or any form of visitor tracking technology.

No Analytics

Third-party analytics services (Google Analytics, Matomo, Plausible, etc.) are not employed on this site.

Minimal Server Logs

Server access logs do not record IP addresses as no IP addresses are ever recorded. Therefore they cannot be logged.

No User Accounts

This site does not require registration, login, or collection of any personal identifiers.

Data Collection

Automatic Data

When you visit this website, your web browser automatically sends certain information to our servers:

  • IP address — All IP addresses are removed from all visitor's request headers. It is therefore never logged.
  • User agent string — Browser and device type information, logged only in access logs
  • Referrer URL — The page you visited before arriving here, logged only in access logs
  • Timestamp — Date and time of request, logged only in access logs

All automatic data is deleted after 7 days and is never shared with third parties or used for profiling purposes.

Voluntary Data

This website has no contact forms, comment systems, newsletter signups, or any mechanism for collecting voluntary user data. There is no way for visitors to submit personal information to us.

Cookies and Storage

We do not use cookies. Your browser will not receive any cookie files from this domain.

If you use Progressive Web App features (offline access via service worker), the following local storage may occur:

Service Worker Cache
Pages you visit may be cached locally in your browser for offline reading. This cache exists only on your device and contains only publicly-available content.
IndexedDB
If available, cached content may be stored in IndexedDB for improved performance. Again, this remains on your device and is accessible only to you.

You can clear this data at any time through your browser settings. It is never transmitted back to our servers.

Third-Party Services

This website intentionally avoids third-party services for the following reasons:

  • Analytics providers — We do not use analytics services and collect no statistics
  • CDN services — Static assets are served directly from our infrastructure
  • Font services — All typography uses system fonts to avoid external requests
  • Social media widgets — No embedded buttons, feeds, or sharing integrations
  • Advertising networks — The site contains no ads and generates no advertising revenue

External links to other websites (such as Project Gutenberg, academic sources, or related resources) are provided for reference. We have no control over those sites' privacy practices and encourage you to review their policies independently.

Security Measures

The following security measures protect visitor data:

  • HTTPS encryption — All communications use TLS 1.3 with strong cipher suites
  • Security headers — Content Security Policy, X-Frame-Options, and other protective headers are implemented
  • Regular updates — Server software and dependencies are kept current to address vulnerabilities
  • Access controls — Administrative access to server infrastructure is restricted and logged

Legal Disclosure

We reserve the right to disclose log data if legally required by subpoena, court order, or other lawful process. However:

  • We notify affected users of such requests whenever legally permitted
  • We challenge overly broad or unconstitutional demands
  • Given our minimal retention period (7 days), much requested data may already be deleted

Your Rights

Because we collect minimal data, your privacy rights are straightforward:

  • Right to access — You can request copies of any logs containing your IP address within the 7-day retention window
  • Right to deletion — Request immediate removal of any retained data
  • Right to object — Object to any data processing (though processing is already minimal)
  • Right to portability — Receive your data in a machine-readable format

To exercise these rights, contact us using the information below.

Children's Privacy

This website does not knowingly collect any personal information from children. Since no personal data collection mechanisms exist (no forms, no accounts, no tracking), children can browse the public-domain literature without privacy concerns.

Changes to This Policy

We may update this privacy policy periodically. Changes will be:

  • Published on this page with a revised "last updated" date
  • Substantive changes will be announced via the site's RSS feed
  • No version of this policy will retroactively reduce your privacy rights

We encourage periodic review of this page, particularly before engaging with any new site features.

All privacy requests will be handled securely and confidentially.
